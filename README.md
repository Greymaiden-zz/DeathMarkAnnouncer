This addon will broadcast in /yell when you apply Death Mark to a raid boss. You can customize the message and channel by opening DeathMarkAnnouncer.lua with the notepad and editing line 49.

On most bosses you will be opening with Garrote to apply Death Mark but on bleed immune bosses you will have to use Ambush, which is why I would highly recommend a weaponswap macro for both Ambush and Hemo/Sinister Strike. SuperMacro is great for this: https://www.briankoponen.com/supermacro-addon-vanilla-world-warcraft/

That's all there is to it. Enjoy!
